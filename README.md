# HRMS Lite – Full-Stack Assignment

**Live Frontend:** [https://hrms-backend-1-thsj.onrender.com] 

**Live Backend:** [https://hrms-frontened-phi.vercel.app/]

**GitHub:** [https://github.com/Abhisheks0n1/HRMS-Lite]

**Tech Stack**  
- Frontend: React 18 + Axios + React Router  
- Backend: FastAPI + SQLAlchemy + MySQL  
- Deployment: Vercel (FE) + Railway (BE + MySQL)

**All Features Implemented**  
- Employee CRUD (Add ,Get List, Update, Delete)  
- Attendance mark + view with date filter  
- Dashboard summary  
- Total present days per employee  
- Professional UI, loading/empty/error states, validations, proper HTTP codes

**Local Run**  
Backend: `uvicorn app.main:app --reload`  
Frontend: `npm start`
